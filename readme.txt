== Description ==
Using VW Files and Notes Downloader you get an option in admin to easily create notes with titles and attached file to it.

Just insert shortcode like [VW_NOTES_AND_FILES_DOWNLOADER]

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `vw-notes-and-files-downloader.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place shortcode: '[VW_NOTES_AND_FILES_DOWNLOADER]' without the inverted commas in your page or post for download files.
